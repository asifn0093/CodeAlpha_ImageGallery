# Image Gallery

An interactive image gallery built with HTML, CSS, and JavaScript.

## Features

- **CSS Grid layout** — responsive gallery with auto-fill columns
- **Category filters** — All, Nature, Architecture, Food, Travel
- **Lightbox view** — click any image for full-size view with prev/next navigation
- **Keyboard support** — Arrow keys (← →) to navigate, Escape to close lightbox
- **Hover effects** — smooth scale, shadow, and overlay transitions
- **Responsive** — adapts to desktop, tablet, and mobile screens
- **Smooth animations** — fade-in lightbox, zoom effect, hover transitions

## File Structure

```
Image Gallery/
├── index.html          # Main page
├── css/style.css       # Styling & responsive design
├── js/script.js        # Gallery logic, filters, lightbox
├── images/             # Gallery images
└── README.md
```

## Usage

Open `index.html` in any modern browser. No server or build tools required.
